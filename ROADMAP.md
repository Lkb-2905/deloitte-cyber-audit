# Roadmap

## Court terme (1-2 semaines)
- Ajouter un README par module avec exemples d'entree/sortie.
- Ajouter un mode `--output` pour tous les scripts (csv/json/html).
- Ajouter des exemples de rapports et captures.

## Moyen terme (1-2 mois)
- Centraliser la configuration (fichier `config.json` par module).
- Ajouter un jeu de tests rapides (pytest) pour les fonctions principales.
- Ajouter un mini-dashboard (static) qui liste les sorties de demo.

## Long terme
- Packager les modules (pyproject.toml) pour reutilisation.
- Ajouter un pipeline CI (lint + tests).
